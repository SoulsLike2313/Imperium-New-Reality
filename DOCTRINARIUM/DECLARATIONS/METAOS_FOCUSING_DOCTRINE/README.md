This directory contains the NewGen doctrine that defines IMPERIUM as a MetaOS focusing layer and enforces organ ownership for internal functions.

## Files

- `METAOS_FOCUSING_DOCTRINE_OWNER_RU_V0_1.md`: owner-facing RU doctrine summary.
- `metaos_focusing_doctrine_v0_1.json`: machine-readable doctrine declaration.
- `organ_function_ownership_law_v0_1.json`: internal-function ownership law.
- `external_force_routing_law_v0_1.json`: external dependency routing law.
- `task_through_organ_core_law_v0_1.json`: mandatory core organ task path law.

## Scope

- Does not claim production autonomy or full external adapter completion.
