# React IDE V0.1

This package hosts the React + TypeScript IDE-shaped projection surface for Agent IDE.

## Local usage

```powershell
cd E:\IMPERIUM\IMPERIUM_NEW_GENERATION\AGENT_IDE\REACT_IDE
npm.cmd install
npm.cmd run dev
```

## Build for web-shell projection

cd E:\IMPERIUM\IMPERIUM_NEW_GENERATION\AGENT_IDE\REACT_IDE
npm.cmd install
