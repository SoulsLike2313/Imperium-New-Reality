# IMPERIUM CLI Heraldry Policy V0.1

## Purpose
Define original heraldry for terminal organ agents.

## Crest Design Principles
- readable in monospaced terminal;
- symbolic, minimal, and reproducible in text;
- independent from external brand assets;
- tied to organ mission and command responsibility.

## Mandatory Components
- crest purpose statement;
- token set and symbolic dictionary;
- per-organ crest registry;
- ASCII examples;
- SVG-safe forward path (tokenized geometry intent only).

## Officio Agentis Direction
Officio crest concept uses original symbols:
- sealed contract;
- double key;
- ledger eye;
- permission ring;
- focus beam.

## Forbidden
- copied Warhammer/Aquila motifs;
- copied Kilo/VS Code logos;
- external copyrighted crests;
- direct franchise icon tracing.
