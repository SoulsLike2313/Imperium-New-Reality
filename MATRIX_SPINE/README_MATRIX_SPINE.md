# IMPERIUM Matrix Spine

Status: `CANDIDATE_V0_1`

Matrix Spine is the routing and measurement layer for Ghost_Evolve V2. It lets Inquisitor, GPT/Logos and Servitor use the same rule base.

## What belongs globally

- index;
- shared vocabulary;
- evidence levels;
- candidate schema references;
- routing to organ-owned matrices.

## What does not belong globally

Detailed enforcement should live under the owner organ:

- Doctrinarium: law/canon/admission;
- Officio: role/language/behavior;
- Astronomicon: task route/questions/stage;
- Administratum: evidence/session/continuity/file atlas;
- Mechanicus: tools/scripts/replay/validators;
- Inquisition: fake-green/dirty/provenance/red-team;
- Strategium: weights/roadmap/next strike;
- Schola: lessons/KPD/pattern reuse.
