# Organ Responsibility Map (Task Volume + Stage Checkpoint)

| Organ | Responsibility |
|---|---|
| Strategium | Owns task volume class, risk, priority, and split recommendation. |
| Astronomicon | Owns route map and stage map. |
| Administratum | Owns checkpoint ledger, diff/evidence bundle, owner decision records. |
| Officio Agentis | Owns execution contract enforcement for volume and stage gates. |
| Inquisition | Detects fake green, skipped requirements, and missing evidence at stage boundaries. |
| Mechanicus | Owns schemas, validators, builders, and CLI tools. |
| Schola Imperialis | Converts failures into lessons and regression examples. |
| Doctrinarium | Enforces doctrine/law: no fake PASS, evidence before pass, no scope bypass. |

Throne and Custodes are out of scope.
