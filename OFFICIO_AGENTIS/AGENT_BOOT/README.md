# Officio Agent Boot V0.1

This directory defines minimal role and response discipline for NewGen task startup.

## Purpose

Move repeatable execution discipline out of oversized taskpacks and into stable organ-owned contracts.

## Boot Inputs

1. `IMPERIUM_NEW_GENERATION/AGENTS.md`
2. Doctrinarium preflight output
3. `officio_role_contract_v0_1.json`
4. Officio execution/language/final response contracts
5. taskpack for concrete scoped work

## Files

- `officio_role_contract_v0_1.json`: machine-readable role contract.
- `servitor_execution_contract_v0_1.md`: mandatory execution order and stop conditions.
- `owner_facing_language_contract_v0_1.md`: Russian owner-facing live communication rule.
- `final_response_contract_v0_1.md`: required final response shape.
- `officio_ack_template_v0_1.json`: baseline acknowledgment schema.
- `TOOLS/officio_boot_ack_v0_1.py`: tool to generate an Officio boot ack artifact.

## Scope Boundary

This package is a minimal boot protocol.

Not proven:

- full organ intelligence;
- automatic enforcement across all agents;
- live organ dialogue;
- Owner Verdict Needed live button;
- production autonomy.
