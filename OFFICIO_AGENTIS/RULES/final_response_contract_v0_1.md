# Final Response Contract V0.1

## Owner-Facing Format

Final response must include:

1. Step name
2. Full path to report/bundle
3. Verdict
4. Short Russian owner comment
5. Commit/push HEAD if performed
6. Worktree clean yes/no
7. Next allowed task

## Quality Clauses

- Do not claim PASS without evidence paths.
- Include concrete blockers for STOP or WARN outcomes.
- Keep summary compact and operational.
- Do not claim global readiness outside scope.

## Machine Artifact Language

Machine artifacts (JSON, scripts, schema keys) remain English-safe UTF-8.
