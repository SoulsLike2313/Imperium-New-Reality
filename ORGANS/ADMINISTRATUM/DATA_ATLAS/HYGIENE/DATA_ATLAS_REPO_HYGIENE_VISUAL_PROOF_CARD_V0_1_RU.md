# Data Atlas Card — Repo Hygiene Visual Proof v0.1

## Назначение

Карточка показывает, что `v0.9.0` repo hygiene classification имеет не только machine report, но и owner-facing смотровую.

## Что должно быть видно

- Files scanned.
- Action required.
- Lane counts.
- Severity counts.
- Top review organs.
- Пути к machine report, lane queue, SQLite index.
- Прямое предупреждение: classification не удаляет source.
