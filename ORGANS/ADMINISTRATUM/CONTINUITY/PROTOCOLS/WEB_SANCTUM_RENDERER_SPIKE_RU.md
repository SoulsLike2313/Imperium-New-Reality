# Web Sanctum Renderer Spike V0.4.3

Этот protocol фиксирует смену визуальной стратегии:

- Tk launcher остается fallback.
- Premium IDE shell переносится в web renderer.
- Backend truth остается Python/Imperium CLI/continuity.
- Web слой получает состояние через read-only JSON snapshot.
- Playwright становится обязательным каналом visual/e2e acceptance.

Safety:

- no real servitor execution;
- no live LLM;
- no unsafe shell;
- no trading execution;
- H-contour first.

Acceptance:

- визуально лучше Tk V0.4;
- плавность лучше на глаз, не только по FPS;
- Mechanicus/Departments модель не путается;
- Playwright smoke проходит;
- owner принимает направление до переноса в main.
