## Extracted pass criteria

- Imperium must become an engineering base for freelance tasks: find/receive work, register task, decompose, execute, test, deliver, support.
- Human-in-loop is allowed and expected: owner and manager can help while protocols mature.
- Sanctum should become a single operator window: no prompt copying, no constant terminal restarts, no fragmented control.
- Taskpack creation must extract pass criteria, forbidden actions, missing info, deliverables, evidence, presentation, support path.
- System must be honest about missing knowledge/tools and ask for research/gaps instead of pretending.
- UI/UX is a core product feature, not decoration. It must be beautiful, smooth, readable, and pleasant.
- Mechanicus becomes the reference organ before cloning standards to other organs.
- Trading department is specialized and must stay gated: research/paper first, no live execution without explicit owner LIVE gate.
