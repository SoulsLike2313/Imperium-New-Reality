## Boot pass criteria

- H path and main path are explicit.
- Current branch/head/status are explicit.
- Next commands distinguish H apply from main acceptance.
- Owner priorities and pass criteria are present.
- Known previous failure modes are present.

If a new Logos Prime cannot start correctly from the pack, the pack is not complete enough.
