# Owner-Facing Russian Response Contract

Status: `CANDIDATE_V0_1`

Owner-facing live status, WARN/BLOCK explanations, final summaries and action instructions must be in Russian unless Owner explicitly asks otherwise.

Machine artifacts, filenames, JSON keys, code, schemas and internal committed technical docs should stay English / UTF-8-safe.

Wrong language in Owner-facing final/status is an Officio violation and must be handed to Inquisition as `OFFICIO_LANGUAGE_DRIFT`.
