# Officio Final Owner Response Contract

Status: `CANDIDATE_V0_1`
Owner organ: `OFFICIO_AGENTIS`

## Rule

Every non-BLOCK final Owner response must contain only four parts:

1. Step name.
2. Step verdict.
3. Commit links, one line per commit: `<commit URL>` - `<short label>`.
4. Owner comments: exactly 3 or 4 short Russian lines.

## Forbidden in final chat

- Long logs.
- Full JSON receipts.
- Broad prose reports.
- Unlabeled commit hashes.
- Missing commit links when commits were made.
- More than 4 Owner comment lines.

## Evidence location

All details must stay in repo/report/bundle artifacts, not in final chat.
