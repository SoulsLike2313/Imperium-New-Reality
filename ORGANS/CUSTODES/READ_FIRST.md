# Custodes Read First

Read this before using Custodes as a task authority.

Custodes is a narrow guard organ. Its job in the current core shape is to verify that required organ homes and organ-life metadata exist, and to record incomplete metadata as warnings or blockers.

For this task, Custodes must not approve physical migration, quarantine active use, or full repository cleanup. Those decisions remain with Administratum, Inquisition, and explicit task acceptance gates.
