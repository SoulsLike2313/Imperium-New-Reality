# Mechanicus / Inquisition Observability Rule V0.1

Любой tool-like script без паспорта является не ошибкой сам по себе, а Inquisition finding типа `NO_TOOL_PASSPORT`.

Mechanicus отвечает за:

- tool passport;
- language reason;
- validation recipe;
- when-to-use / when-not-to-use;
- action registry eligibility.

Inquisition отвечает за:

- обнаружение отсутствующего паспорта;
- severity;
- evidence;
- blocking gate;
- owner-readable recommendation.

Data Atlas отображает этот вывод рядом с файлом.
