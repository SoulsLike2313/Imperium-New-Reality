# Mechanicus Forge Client V0.6 Soft RU Skin

## Что меняется

- Более мягкая подача текста.
- Цветовая палитра: бордо, золото, ледяная сталь.
- Фиолетовый "дух машины" проходит по ключевым строкам и акцентам.
- Все surfaces остаются прежними: меняется только skin + tone.

## Важно

Реальный шрифт терминала из Python/Textual поменять нельзя: его задаёт сам терминал.
Но можно сделать визуал мягче:
- меньше кричащего upper-case;
- мягче контраст;
- спокойнее рамки;
- благороднее палитра.
