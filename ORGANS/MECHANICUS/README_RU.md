# MECHANICUS ORGAN V0.4

Mechanicus is the first exemplar organ for Imperium New Reality.

It is responsible for:

- state machines instead of endless conditional chains;
- schema-first tool safety;
- DB / API / RAG / search corridors behind explicit gates;
- dry-run, receipt, rollback and test evidence for tool use;
- UI standards for readable, smooth, operator-safe panels.

Safety: this folder is a foundation only. It does not enable live execution, unsafe shell, live LLM, external API keys or trading execution.
