# Logos-Prime handoff contract

The continuity pack must prepare a new Logos-Prime/new chat to enter the current project state without guessing.

## Required handoff content

- Latest accepted HEAD.
- Current branch and remote sync truth.
- Latest task chain and verdicts.
- Active caps and warnings.
- Current next recommended task.
- Key organ state summary.
- Important evidence/report paths.
- What not to assume.
- What to read first.
- Owner-facing concise Russian summary.
- Machine-readable English summary.

## Role-entry requirement

Officio Agentis owns the Logos-Prime role-entry/language contract. The handoff pack should reference Officio authority when present, without inventing a role contract if source files are absent.
