# Skill terminology and ownership

A Skill is a reusable, validated operational hand of an organ or agent. It is not merely an external launcher.

Ownership model:

- Astronomicon owns Taskpack Registration Skill.
- Administratum owns Continuity Pack / Logos-Prime Handoff Skill.
- IDE may call both Skills later, but IDE must not become the truth source.
- Mechanicus validates scripts and tool compatibility.
- Inquisition audits fake-green, fake-continuity, stale context and leak risk.
- Officio Agentis owns role-entry and runtime Owner-facing language contracts.
- Schola Imperialis stores reusable learning cards.
