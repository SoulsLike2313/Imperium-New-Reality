# Open Astronomicon TUI and register this PC task

Use this taskpack to test the new Astronomicon Taskpack Registration Skill on PC contour.

Expected manual route:

1. Open Astronomicon TUI.
2. Choose Taskpack Registration Skill or Register Taskpack.
3. Select this ZIP from Downloads:

`TASKPACK_NEWGEN_ADMINISTRATUM_CONTINUITY_PACK_SKILL_AND_LOGOS_HANDOFF_BRIDGE_PC_V0_1.zip`

4. Choose contour: `PC`.
5. Confirm registration.
6. Verify admission PASS and resolver PASS/PASS_WITH_WARNINGS.
7. Copy the launch card message to Servitor:

`start task`

Fallback if TUI entrypoint cannot be found: use raw Astronomicon intake/resolver and record the TUI gap as a warning, not as a clean PASS.
