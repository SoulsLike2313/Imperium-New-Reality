# Officio Owner Language Routing

Owner-facing runtime response language: Russian.

Machine artifacts, repository artifacts, JSON, schemas, scripts, logs, and canonical report files: English, UTF-8, no BOM.

Russian is allowed only in the final chat response to Owner or in an explicitly named Officio Owner-facing summary artifact if the task creates one.

If a Russian Owner summary is created, it must be marked as Owner-facing localization and must not be used as a canonical machine proof file.

The final Owner response must use the 4-part contract:

1. Step name
2. Step verdict
3. Commit links / commit identifiers with labels
4. Owner comments: 3-4 short Russian lines
