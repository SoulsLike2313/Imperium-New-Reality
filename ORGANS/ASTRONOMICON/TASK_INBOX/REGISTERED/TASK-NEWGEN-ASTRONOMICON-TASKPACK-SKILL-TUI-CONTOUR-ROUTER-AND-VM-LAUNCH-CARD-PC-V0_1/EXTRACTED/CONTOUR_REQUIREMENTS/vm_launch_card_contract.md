# VM Launch Card Contract

For VM2 and VM3, successful registration is not complete unless the target VM shows a launch-card terminal or a limitation receipt explains why it could not be shown.

Required launch card content:

```text
IMPERIUM TASK LAUNCH CARD

STEP:
<human readable step name>

TASK_ID:
<task id>

REGISTERED TASK PATH:
<path on target contour>

TASKPACK:
<registered TASKPACK.zip path>

CHAT MESSAGE TO SERVITOR:
start task

COPY ONLY THIS INTO SERVITOR CHAT:
start task
```

The launch card must be generated on the target contour after intake/resolver PASS or PASS_WITH_WARNINGS.
