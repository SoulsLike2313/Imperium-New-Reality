# DAILY OPS SHELL BLUEPRINT

## Goal

The Owner opens one station surface and sees what to do next.

## Daily board sections

- System truth;
- Current task;
- Current expected task;
- Agent roster;
- Taskpack manager snapshot;
- Launch and handoff card state;
- Lifecycle;
- Safety;
- Dirty state;
- Git closure;
- Next action.

## Guided workflow

Start day
-> inspect system
-> create or select task
-> build taskpack
-> validate
-> dry-run register
-> review live promotion
-> copy handoff
-> watch reports
-> close git
-> next task.
