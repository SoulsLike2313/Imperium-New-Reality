# Что дать Servitor после регистрации taskpack в Astronomicon

После того как Astronomicon TUI зарегистрирует ZIP и покажет task_id, дай Servitor:

```text
TASK_ID: TASK-NEWGEN-STAGE3-FIRST-REAL-USE-GHOST-EVOLVE-MICROPILOT-PC-V0_1
start task
```

Не давай длинный prompt.
Цель — проверить, что Servitor сам найдёт задачу через Astronomicon.
