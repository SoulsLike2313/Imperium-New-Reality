# Ghost_Evolve V2 Stage 3.1 Teaching Rules

Every Stage3 pain must become durable system learning:
- missing template -> bootstrap preflight criterion;
- UTF-8 BOM -> encoding guard and fixture;
- fixture runner picked as launcher -> owner launcher policy;
- intake blocked before ZIP read -> preflight-before-intake policy;
- root/flat ZIP compatibility issue -> taskpack format compatibility matrix item;
- PENDING finalization fields -> external finalization cleanup rule.
