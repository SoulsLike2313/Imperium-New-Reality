# Officio Owner Language Routing

Owner-facing runtime output may be in Russian only under Officio Agentis authority.

Canonical repository artifacts, taskpack internals, JSON, schemas, scripts, and machine receipts must be English, UTF-8, no BOM.

If Russian Owner-facing summaries are stored as artifacts, they must be explicitly marked as Owner-facing Officio output and must not be treated as machine contract text.
