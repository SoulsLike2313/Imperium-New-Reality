# Final response contract requirements

Register this as an Officio Agentis controlled contract.

## Mandatory final response form for every non-BLOCK worker result

1. Step name.
2. Step verdict.
3. Commit links, one line per commit:
   - `<commit URL>` - `<short label explaining what the commit carries>`
4. Owner comments: exactly 3 or 4 short Russian Owner-facing lines.

## Forbidden in final chat

- Long logs.
- Full JSON receipts.
- Broad prose reports.
- Unlabeled commit hashes.
- Missing commit links when commits were made.
- More than 4 Owner comment lines.

## Evidence location

All details must be in repo/report/bundle.
