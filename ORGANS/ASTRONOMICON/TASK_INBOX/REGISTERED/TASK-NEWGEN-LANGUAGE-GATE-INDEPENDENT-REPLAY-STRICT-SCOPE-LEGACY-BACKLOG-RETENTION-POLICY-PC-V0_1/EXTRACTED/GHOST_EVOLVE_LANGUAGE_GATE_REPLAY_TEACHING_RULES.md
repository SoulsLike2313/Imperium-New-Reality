# Ghost_Evolve Language Gate Replay Teaching Rules

Every replay, backlog, stale-truth, or retention issue must become durable system learning.

Examples:
- Legacy language hit -> backlog item and owning organ candidate.
- Strict-scope clean but global warnings remain -> no clean PASS, create phased remediation plan.
- Stale hard-red-team repo truth -> head taxonomy update.
- Registered taskpack footprint grows -> retention policy.
- Taskpack ZIP contains large nested payload -> hash/quarantine recommendation.
- Admission gate cannot show strict/global distinction -> Astronomicon improvement candidate.

Each learning item must include:
- source problem;
- evidence path;
- owning organ;
- affected route;
- cap mapping;
- future prevention;
- script-first remediation path.
