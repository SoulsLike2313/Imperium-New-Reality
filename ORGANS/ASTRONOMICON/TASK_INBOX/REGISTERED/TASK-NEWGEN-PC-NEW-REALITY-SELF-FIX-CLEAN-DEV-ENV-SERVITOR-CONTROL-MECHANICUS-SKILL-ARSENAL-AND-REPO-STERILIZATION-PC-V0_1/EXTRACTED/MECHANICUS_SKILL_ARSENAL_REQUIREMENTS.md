# Mechanicus Skill Arsenal Requirements

Mechanicus must register candidates, not install tools blindly.

Candidate states:

- `CANDIDATE`
- `SANDBOX_READY`
- `SANDBOX_TESTED`
- `ADMITTED`
- `CANON`
- `QUARANTINE`
- `REJECTED_FOR_NOW`

Canonization rule:

A candidate cannot become canon until 10 successful task cycles with receipts.

Seed candidate cards:

1. Graphify
2. SearXNG
3. Crawl4AI
4. Public APIs registry
5. State-machine discipline / XState pattern study
6. Wasmer sandbox candidate
7. Antares / database visibility candidate

No external installation is required in this task.

Each candidate card must include:

- candidate_id
- name
- category
- owner_organ
- purpose
- first_safe_use_case
- risk
- admission_mode
- install_status
- receipt_requirements
- canon_after_successful_cycles
