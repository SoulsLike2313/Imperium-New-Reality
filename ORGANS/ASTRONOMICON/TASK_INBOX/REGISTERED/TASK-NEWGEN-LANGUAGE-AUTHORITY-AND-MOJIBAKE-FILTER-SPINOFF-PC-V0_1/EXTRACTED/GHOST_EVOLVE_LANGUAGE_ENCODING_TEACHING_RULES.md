# Ghost_Evolve Language and Encoding Teaching Rules

Every language or encoding failure must become durable system learning.

Examples:
- Cyrillic in canonical internal artifact -> Officio authority rule and Inquisition cap.
- UTF-8 BOM -> encoding fixture and repair path.
- Mojibake signature -> Inquisition signature update.
- Runtime owner-facing Russian stored as policy text -> Officio route correction.
- Localization resource not marked -> localization exception matrix item.
- Taskpack root file violates English-only policy -> Astronomicon admission gate update.

Each learning item must include:
- source problem;
- evidence path;
- owning organ;
- affected task route;
- cap mapping;
- future prevention;
- script-first remediation path.
