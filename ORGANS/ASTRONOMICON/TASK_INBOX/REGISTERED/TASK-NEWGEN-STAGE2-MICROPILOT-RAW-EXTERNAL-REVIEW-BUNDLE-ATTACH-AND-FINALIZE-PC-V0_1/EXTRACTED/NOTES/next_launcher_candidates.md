# Next Launcher Candidates

Task ID:

`TASK-NEWGEN-ASTRONOMICON-VM3-TASKPACK-LAUNCHER-AND-IDE-HANDOFF-BRIDGE-V0_1`

Purpose:

Create a short, stable PC-side launcher that sends a taskpack to VM3, runs Astronomicon intake, blocks on admission failure, runs resolver, blocks on resolver failure, and opens a VM3 launch-card terminal with STEP, TASK_ID, registered path, taskpack path, and exact `start task` message.

This is Astronomicon-owned. IDE can call it later.
