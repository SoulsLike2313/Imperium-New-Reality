# Taskpack Self-Start Read Order Contract

Status: `CANDIDATE_V0_1`

Owner should be able to provide a ZIP and write `start task`.

The taskpack must include:

- `000_START_TASK_READ_ORDER.md`
- task id and scope
- target contour
- expected input files
- required organ read-order
- acceptance gates
- forbidden claims
- required output bundle structure

The taskpack must not replace Officio/Doctrinarium/Inquisition/Mechanicus authority.
