# Git Dirty Actions Guide

Status: CANDIDATE_V0_1

Open classification:

- python ORGANS/IMPERIAL_IDE/SHELL/imperial_ide_cli.py dirty-classifier
- python ORGANS/IMPERIAL_IDE/SHELL/imperial_ide_cli.py git-closure

Dirty Classifier v2 separates current task report artifacts, Astronomicon registry/taskpack admission evidence, old unrelated report artifacts, runtime/generated taskpack paths, local config, secret risk, stage candidates, keep unstaged, and owner decision needed.

Git Closure prints exact dirty count, classified table, push_allowed_state, recommended commands, paths that can be staged after validation, and paths that must not be staged.

Deletion or quarantine is not performed without a separate owner-approved batch task.
