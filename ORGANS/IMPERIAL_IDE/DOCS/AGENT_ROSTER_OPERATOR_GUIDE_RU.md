# Agent Roster Operator Guide

Статус: CANDIDATE_V0_1

## Primary view

Основная модель агентов берется из ORGANS/IMPERIAL_IDE/AGENTS/agent_registry.json и servitor_roster.json.

Required servitors:

- PRIME
- ASTRONOMICON
- MECHANICUS
- ADMINISTRATUM
- INQUISITION
- STRATEGIUM
- DOCTRINARIUM
- OFFICIO_AGENTIS
- SCHOLA_IMPERIALIS
- WARP
- METAOS
- IMPERIAL_IDE

Legacy Alpha/Beta/Gamma capsules остаются candidate dry-run моделью и не являются primary roster.
