# IDE Build Ladder

## Stage 1: Foundation

Contracts, schemas, registries, workspace model, and Mechanicus bridge. This task completes Stage 1 only.

## Stage 2: Local TUI Or Web Shell

Build a minimal interface for taskpack control, registry browsing, report browsing, receipt inspection, and dry-run Mechanicus calls.

## Stage 3: Validation Replay

Add replay buttons for JSON parse, Python compile, Mechanicus doctor, Astronomicon discovery smoke, and scope scans.

## Stage 4: Product Workspace

Add owner request-to-product workspace panels with tasks, artifacts, receipts, and pushed closure.

## Stage 5: Controlled Real Tool Execution

Add sandboxed, allowlisted, receipt-producing real execution only after owner approval.
