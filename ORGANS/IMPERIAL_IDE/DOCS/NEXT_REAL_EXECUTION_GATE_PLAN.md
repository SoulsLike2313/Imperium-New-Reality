# Next Real Execution Gate Plan

Real servitor or LLM execution is outside this task. A future gate must provide all of the following before activation:

1. A named backend and credential source that is never committed.
2. A strict per-agent allowlist and argument schema.
3. Owner confirmation for high-risk or mutating actions.
4. Timeout, cancellation, concurrency, and rate-limit controls.
5. Prompt, tool-call, output, and cost receipts with secret redaction.
6. Replay-resistant task identity and lifecycle linkage.
7. Inquisition fake-green tests and Mechanicus policy validation.
8. A kill switch that defaults to blocked after restart.

Until that task is admitted and validated, `live_llm_backend_enabled`, `real_servitor_execution_enabled`, `unsafe_shell_available`, and `arbitrary_shell_allowed` remain false.
