# Imperial IDE Workbench

The Workbench is an owner-facing candidate surface over the active Imperial IDE
control shell. It reads the current repository through `IMPERIUM_ROOT` or root
discovery and falls back to explicitly labelled sample data.

Current boundary:

- TUI smoke is available.
- GUI structural smoke is available without opening a window.
- Interactive Tkinter GUI remains a Windows candidate until owner smoke.
- Servitor capsules are one-shot dry-run models only.
- Real process dispatch and persistent supervisors are blocked.

Use `ORGANS/IMPERIAL_IDE/run_imperial_workbench.ps1 -Surface smoke` for the
non-interactive smoke path.
