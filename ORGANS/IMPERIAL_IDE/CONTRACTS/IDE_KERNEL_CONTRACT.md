# IDE Kernel Contract

The New Reality repository is the kernel for the future Imperial IDE.

## Kernel Responsibilities

- expose governance, taskpack, organ, report, receipt, and tool registry surfaces;
- preserve root truth through `AGENTS.md` and `ORGANS/_CORE_GOVERNANCE/`;
- keep task execution evidence in reports and git receipts;
- reject full-IDE-complete claims unless a dedicated implementation task proves them.

## IDE Responsibilities

- read kernel state without rewriting unrelated files;
- request tools through Mechanicus;
- expose taskpack control, report browsing, receipt inspection, validation replay, and product workspaces;
- keep every mutating action inside a task scope.
