# WARP Runtime v0.6

The WARP zone is a temporary work contour created outside the source repo, preferably by git worktree and with copy fallback.

Cycle:

1. Create WARP.
2. Register/activate Astronomicon taskpack.
3. Start Work.
4. Work in WARP.
5. Validate WARP.
6. Build Administratum report bundle.
7. Owner decides promotion.

Generated task forms are exported to `IMPERIUM_LOCAL_HANDOFF/TASK_FORMS`, not into the source tree.
