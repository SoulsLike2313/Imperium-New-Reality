# Первый запуск MetaOS

Запустите безопасный smoke из корня репозитория:

```powershell
& .\ORGANS\IMPERIAL_IDE\run_metaos_smoke.ps1
```

MetaOS использует только детерминированные заглушки. Реальный LLM, секреты и
фоновый runtime не включены. Показатели токенов и стоимости являются оценкой
candidate-модели, а не данными реального биллинга.
