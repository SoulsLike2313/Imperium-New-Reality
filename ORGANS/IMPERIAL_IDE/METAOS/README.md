# MetaOS Orchestration Candidate

MetaOS is a script-first orchestration candidate for deterministic routing,
one-shot thin servitors, organ chronicles, and Administratum bundle gates.

All model runners are deterministic stubs. Token and cost values are candidate
estimates, not live billing measurements. Live LLM backends, secrets, external
calls, and persistent runtimes are disabled.

Use `ORGANS/IMPERIAL_IDE/run_metaos_smoke.ps1` for the integrated smoke path.
