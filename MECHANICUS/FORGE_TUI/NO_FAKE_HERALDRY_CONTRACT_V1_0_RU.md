# Mechanicus Forge Client V1.0 — No Fake Heraldry Contract

## Mode

READ_ONLY_NO_FAKE_HERALDRY.

## Decision

No more fake ASCII heraldry pretending to be real emblems.

## Preserved

- read-only mode;
- no install actions;
- no registry writes;
- no evidence writes;
- no server;
- action sockets sealed;
- RU index layer;
- modular surfaces.

## Future visual law

True heraldry belongs to enhanced visual mode, not pure TUI.
Pure TUI may show only sigil badges or text seals.
