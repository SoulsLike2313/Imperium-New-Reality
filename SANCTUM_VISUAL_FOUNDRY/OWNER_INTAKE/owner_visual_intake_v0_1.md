# Owner Visual Intake V0.1

## Slice
Mechanicus console in Sanctum / Visual Foundry

## Primary feeling
A living organ of the IMPERIUM brain.

## First glance priority
- living neural brain / second mind form
- active organ zone
- real current activity
- right contextual organ panel

## Style mix
- 60% black/red Mechanicus forge
- 20% dark purple nebula metallic
- 20% clean sci-fi glass/holograph

## Visual intensity
5-7 out of 10

## Layout mode
Split screen:
- left/center: brain and organ zones
- right: animated current organ panel
- brain pulses while system life is shown

## Mechanicus panel purpose
- report current work
- show approved tools
- run tools check
- verify new contour toolchain
- confirm Windows/Ubuntu tool readiness

## Sanctum-level actions
- send prompt/message to VM
- fetch bundle
- build continuity
- run audit
- inspect dirty state
- contour verification

## Hard forbidden
- visual/backend desync
- raw terminal as primary screen
- generic mainstream dashboard
- fake pass or fake state
- useless decorative animation
- weak attention hierarchy
- unreadable dense layout
- uncontrolled buttons without operational meaning
- mixing Sanctum shell controls and internal TUI controls without separation
- fake ambient life with no relation to system state unless clearly labeled ambient-only

## Acceptance statement
Accepted when the Owner sees a beautiful sci-fi Warhammer-like IMPERIUM holograph neuro second-mind form and Mechanicus reports its work clearly and truthfully.
