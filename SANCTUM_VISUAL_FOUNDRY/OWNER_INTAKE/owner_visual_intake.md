# Owner Visual Intake

## Slice

Mechanicus operator console slice

## Main goal

Create a high-readability, premium-feeling operator console that shows truth, current work, command access, tool capability, and latest evidence without turning raw logs into the primary surface.

## Must-have regions

- Top truth strip
- Work zone
- Command rail
- Tool registry summary
- Footer summary cards
- Raw/technical mode as secondary panel only

## Style words

- Dark metallic
- Mechanicus
- Operator console
- Cyan / red / amber / green semantic accents
- Strong hierarchy
- Clean grouping

## What must not happen

- Raw terminal as primary screen
- Tiny unreadable text
- Generic card soup with weak hierarchy
- Scope contamination outside `IMPERIUM_NEW_GENERATION`

## Reference notes

- Target should feel like a real operator station, not a dump viewer
- Truth and actionable controls must be visually first
- 1366x768 must remain readable without layout collapse
- 1920x1080 should preserve density and balance

## Priority viewport

1366x768 and 1920x1080
