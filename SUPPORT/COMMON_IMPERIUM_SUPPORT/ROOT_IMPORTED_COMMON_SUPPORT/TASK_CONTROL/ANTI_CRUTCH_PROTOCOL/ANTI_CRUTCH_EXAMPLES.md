# Anti-Crutch Examples V0.1

## Canonical Officio Rule
A language/response behavior rule references Officio contracts/checkers and is not taskpack-only.

## Taskpack-Only Officio Workaround
A language rule appears only as a local prompt note and asks no promotion to Officio owner artifacts.

## Canonical Mechanicus Rule
A tool/checker rule points to Mechanicus/CLI/registry ownership.

## Taskpack-Only Tool Workaround
A text says "just tell Servitor to use random tool" without registry/checker ownership.

## Canonical Doctrinarium Rule
A compliance law is promoted to doctrine/gate artifacts.

## Prompt-Only Law Workaround
A law is declared as prompt text only with no doctrine promotion.
