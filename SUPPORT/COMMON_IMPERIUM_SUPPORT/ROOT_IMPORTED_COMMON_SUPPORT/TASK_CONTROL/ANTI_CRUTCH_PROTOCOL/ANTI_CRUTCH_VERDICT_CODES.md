# Anti-Crutch Verdict Codes V0.1

## Core
- `NO_CRUTCH_FOUND`
- `WARN_TASKPACK_ONLY_WORKAROUND`
- `WARN_TEMPORARY_WORKAROUND_ACCEPTED_WITH_DEBT`
- `BLOCKED_CANONICAL_AUTHORITY_BYPASSED`
- `BLOCKED_ORGAN_AUTHORITY_MISSING`

## Promotion Debt
- `DEBT_PROMOTE_TO_OFFICIO`
- `DEBT_PROMOTE_TO_DOCTRINARIUM`
- `DEBT_PROMOTE_TO_MECHANICUS`
- `DEBT_PROMOTE_TO_INQUISITION`
- `DEBT_PROMOTE_TO_ADMINISTRATUM`
- `DEBT_PROMOTE_TO_ASTRONOMICON`
- `DEBT_PROMOTE_TO_STRATEGIUM`
- `DEBT_PROMOTE_TO_SCHOLA`

## Decision Rule
- Blocked codes dominate warns.
- Warn codes dominate `NO_CRUTCH_FOUND`.
- If no warns/blocks are found, return `NO_CRUTCH_FOUND`.
