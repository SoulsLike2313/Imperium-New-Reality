# Agent General Skill Spine V0.1

## Purpose
Provide a shared execution spine for agents to reduce generic behavior and enforce proof-first work.

## Coverage
- read-first discipline;
- scope discipline;
- super skepticism;
- proof records;
- no fake green;
- final owner reporting.

## Required skill passports (minimum set)
1. `agent.read_first_admission.v0_1`
2. `agent.scope_boundary_control.v0_1`
3. `agent.super_skepticism_loop.v0_1`
4. `agent.proof_record_writer.v0_1`
5. `agent.no_fake_green_reviewer.v0_1`
6. `agent.owner_report_compactor_ru.v0_1`
7. `agent.git_truth_guard.v0_1`

## Shared protocol
1. Resolve authority files before mutable action.
2. Bind work to explicit allowed and forbidden paths.
3. Attach evidence plan before execution.
4. Emit machine-readable proof artifacts.
5. Run validator/checker before pass verdict.
6. Distinguish proved vs unknown outcomes.
7. Emit owner report with limitations and next step.

## Forbidden moves
- starting edits without GATE_ACK;
- claiming PASS from plausible-only evidence;
- hiding dirty status or skipped checks;
- deleting generated useful tools without classification;
- crossing forbidden roots.

## Required proof
- before/after git truth;
- proof records JSONL;
- validator report JSON;
- changed files report;
- final receipt.
