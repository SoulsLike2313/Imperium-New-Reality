# No External Logo Policy

This CLI layer and organ crests must stay original.

Forbidden sources:
- Kilo logos, color systems, UI assets, and iconography;
- Warhammer and related franchise symbols;
- VS Code and other vendor logos;
- any external copyrighted crest copied as terminal identity.

Allowed:
- abstract UX principles (mode discipline, slash commands, readable shell);
- original token-based symbols authored inside IMPERIUM.

Verification:
- crest tokens and examples live under this folder;
- changed files list and report must show no imported image/logo assets.
