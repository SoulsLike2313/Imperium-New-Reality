## Root Imported Common Support

The ROOT_IMPORTED_COMMON_SUPPORT subzone is used for physically migrated root-level shared support, contracts, schemas, tools, docs, task-control material, and historical support evidence that should no longer live directly at active root.

Items in this subzone may remain active support only when their migration address-book entry marks active_use_allowed as true.
