# Agent IDE Block Foundation V0.1

Task: `TASK-NEWGEN-AGENT-IDE-DUAL-SURFACE-SELF-VALIDATOR-BLOCK-FOUNDATION-PC-V0_1`

## Purpose

Seed semantic block contracts so desktop and projection surfaces can evolve without mixing:

- truth/data contracts;
- visual skin/material;
- animation policy;
- safety/projection visibility.

## Included

- `block_schema_v0_1.json`
- `block_registry_seed_v0_1.json`
- `layout_manifest_schema_v0_1.json`
- `theme_tokens_v0_1.json`
- `material_registry_v0_1.json`
- `animation_registry_v0_1.json`

## Safety

- Private/local/runtime-sensitive data must never be fully exposed in projection.
- Restriction policy is represented by `safety_level` and `projection_visibility`.
- Projection remains consumer of shared view model, never truth source.
