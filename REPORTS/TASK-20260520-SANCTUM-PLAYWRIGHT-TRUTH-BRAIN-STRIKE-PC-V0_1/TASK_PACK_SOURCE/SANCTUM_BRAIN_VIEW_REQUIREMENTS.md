# Sanctum Brain View Requirements

Minimum implementation target:

1. A central brain-like composition or brain-zone layout.
2. Ten clearly marked zones or nodes:
   - 8 main organs
   - Custodes
   - Throne
3. Zone logic must feel biological + sci-fi.
4. Neural link motifs / signal lines between zones.
5. Smooth hover / selection state changes.
6. Mechanicus zone opens/refocuses to real content.
7. Other zones can be placeholder, but honest.
8. A clear note of truth mode:
   - real
   - placeholder
   - locked
9. If not enough time for advanced 3D, still deliver the strongest 2D/2.5D brain-zone foundation possible.
10. No pass if the center is still just generic boxes with no brain logic.
