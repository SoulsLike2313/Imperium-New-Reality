# Pass Criteria

## Hard pass gate
PASS is allowed only if all of the following are true:

1. Playwright was run and saved evidence exists.
2. Brain-like Sanctum zone system exists visibly.
3. Mechanicus is shown honestly as real connected content.
4. Other organs are marked honestly as placeholder/locked if not real.
5. Fake green audit exists and passes honestly.
6. Improvement plan exists.
7. OSS candidate matrix exists.
8. Performance report exists.
9. Commit/push receipt exists if verdict is PASS.

## Hard fail gate
Must FAIL if any of the following is true:

- No real brain view.
- Brain view exists but does not clearly show organ zones.
- Playwright evidence missing.
- UI claims not backed by truth.
- Mechanicus shown incorrectly.
- Placeholder organs shown as real without backing.
- Performance/animation claims made with no evidence.
- Commit/push claimed but not actually done.
