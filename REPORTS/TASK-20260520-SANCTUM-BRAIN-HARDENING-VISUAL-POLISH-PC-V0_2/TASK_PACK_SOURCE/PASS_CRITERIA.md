# PASS CRITERIA

## Hard PASS requirements
The task may be marked PASS only if ALL of the following are true:

1. Sanctum brain-view no longer places organ cards as a central cluster.
2. Organ cards are visually placed around the outer/peripheral brain zone.
3. Center area is clearly a brain-core / neural communication zone.
4. Real/placeholder/locked truth labeling remains honest.
5. Previous evidence debt is repaired with committed final receipt values.
6. Playwright truth audit for the updated state passes.
7. Final git/HEAD/remote evidence is captured after implementation.
8. Commit and push are completed.

## WARN conditions
Use WARN if any of the following are true:
- visual target improved but not yet fully convincing;
- center neural zone exists but is still weak / static / underpowered;
- animation/performance still not strong enough for hard visual target;
- some evidence is present but secondary polish debt remains.

## FAIL / BLOCK conditions
Use FAIL or BLOCK if any of the following are true:
- organ cards are still centered instead of peripheral;
- center zone is not clearly a neural communication/brain-core zone;
- fake-green or untruthful labels appear;
- Playwright audit regresses or is skipped;
- final receipts remain placeholder/pending;
- commit/push is skipped despite passing scope.
