# Pack Builder Contract

Role pack output is a portable ZIP bundle for role entry.

Required files:

- `ROLE_PACK.md`
- `role_pack.json`
- `RESPONSE_CONTRACT.md`
- `EXECUTION_SETTINGS.md`
- `STOP_CONDITIONS.json`
- `EVIDENCE_POLICY.md`
- `START_MESSAGE.txt`
- `MANIFEST.json`
- `SHA256SUMS.txt`

Pack build must print the ZIP path and manifest path.
