# Officio Shell Contract V0.1

Officio shell is the first executable proof of COMMON_AGENT_CLI.

## Contract Points
1. Uses COMMON_AGENT_CLI command palette with short command-first help.
2. Shows product-grade layout zones: top status, left work, right command panel, bottom event bar.
3. Displays shell boundary message for OS commands typed inside agent shell.
4. Reports renderer truth in `renderer_diagnostic.json`.
5. Shows original Officio crest text and IMPERIUM color accents.
6. Keeps legacy command aliases for backward compatibility.

## Required Runtime Artifacts
- shell transcript
- renderer diagnostic
- color diagnostic
- shell receipt
